VIBE

A token built in public to document how Ethereum actually works.

No promises. No roadmap. No whitepaper.


VIBE is an ERC-20 token deployed to Ethereum mainnet in a single afternoon — hand-rolled in Solidity, no frameworks, no abstractions. The contract, the deployment scripts, and the Uniswap liquidity pool were all built from scratch as a learning exercise.


Every step is documented in this blog post. The code is on GitHub.
